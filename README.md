NamedCharacterConverter
=======================

This library provide you a simple way to convert and encode html special character into words character.

This version only supports:

    '&gt;' => '>',
    '&lt;' => '<',
    '&#39;' => '\'',
    '&amp;' => '&',
    '&quot;' => '"',
    '&nbsp;' => ' ',
    '&aacute;' => 'á',
    '&eacute;' => 'é',
    '&iacute;' => 'í',
    '&oacute;' => 'ó',
    '&uacute;' => 'ú',
    '&ntilde;' => 'ñ',
    '&Aacute;' => 'Á',
    '&Eacute;' => 'É',
    '&Iacute;' => 'Í',
    '&Oacute;' => 'Ó',
    '&Uacute;' => 'Ú',
    '&Ntilde;' => 'Ñ',
    '&uuml;' => 'ü',
    '&Uuml;' => 'Ü',
